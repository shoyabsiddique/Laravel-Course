This is for those Using Vagrant / Homestead - Everyone else disregard this lecture

  1. vagrant box list - this will show the version number and boxes - write down or copy this information
  2. vagrant box update - downloads the and install the updates
  3. vagrant destroy - destroy the old box
  4. vagrant up - This provisions the box and now the changes will take effect


You can purge the old box by doing 

vagrant box remove laravel/homestead --box-version=5.1.0

--------------

This commands could change so make sure you check the Documentations all the time! 

Thanks and hopefully that helps!
